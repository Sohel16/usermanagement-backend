# Node.js Express Backend

A simple **Node.js backend API** built with **Express.js**.  
This project serves as a starter template for RESTful APIs.

---

## 🚀 Features

- RESTful API endpoints
- JSON responses
- Easy-to-extend route structure
- Ready for database integration (MongoDB, PostgreSQL, etc.)

---

## 🛠 Prerequisites

- [Node.js](https://nodejs.org/) v14+
- npm (comes with Node.js)

---
